Minimalistic chrome/chromium plugin to make transmission of passwords from the clipboard to the browser more secure.
IMPORTANT:
- You use this software at your own risk!
- make sure you execute build_permutation.py before using this plugin.
  This generates a new permutation string which is used to encode/decode passwords
FEATURES:
- context menu entry "Insert encoded password" for editable fields.
- Encode a password/Decode a password when opening the context menu somewhere else on the page
